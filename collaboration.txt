### Technische Details

Das Tool ermöglicht sicheres Umbenennen von Symbolen (Typen, Methoden, Properties, etc.) 
über die gesamte Solution hinweg. Nutzt Roslyn's Renamer API für korrekte Referenz-Updates.

**Parameter:**
- `symbolId`: Zu umbenennendes Symbol
- `newName`: Neuer Identifier

**Build-Hinweis:** Projekt benötigt .NET 10.0 SDK (Preview Features: Extension Types)
